O que é a Internet?

No início dos anos 70, com o objetivo de interligar os computadores de seu departamento, a ARPA(Advanced Research and Projects Agency) criou a ARPANET, que conectava quatro universidades norte-americanas.

A partir dos anos 80, outras redes foram interligadas, inclusive o supercomputador da NSF(National Science Foundation), que junto ao ARPANET formava a espinha dorsal da Internet.

No final dos anos 80, empresas privadas começaram a inestir no projeot e, a partir de 1993, a internet deixou de ser exclusivamente acadêmica e passou a ser explorada por todos.

E este tal de HTML?

O HTML (Linguagem de Marcação de HiperTexto) é o "esqueleto" dos sites da web.

É com esta linguagem que o conteúdo de um determinado website é ofertado, informando o navegador sobre como a página deverá ser montada.

Para tal, o HTML faz uso de elementos conhecidos como TAGS que envolvem, marcam ou anexam diferentes partes do conteúdo para que tenham determinado comportamento no navegador.

Tags HTML

As tags do HTML são responsáveis por criar elementos.

Cada uma representa um tipo de elemento no corpo do html, como um título, um parágrafo, uma lista, etc. Também são utilizadas para estilizar o texto, como sublinhar, deixar em negrito ou itálico.

Também, algumas podem receber certos atributos que as alteram de alguma forma.
